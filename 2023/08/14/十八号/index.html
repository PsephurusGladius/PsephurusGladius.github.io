十八号

发表于 2023-08-14 更新于 2023-08-15
回忆录序言。

白昼开始缩短，潮湿的江风藏着一贯的似有似无的腥气，而我的情绪仍然找不到落脚点。

我计划以某个夏天的一场暴雨来作为序言的开头。不知五年级还是六年级的我，骑着那辆过分熟悉的说大不大说小不小的蓝色花纹自行车，在暴烈冲刷着的雨水激荡起的飘忽白雾中全速前行。而多年后偶然谈到痛快淋一场雨的话题，那夜的风声雷声与雨声突然降临而直接笼罩了我的整副躯壳，可内心明明骚动着的我却声称不再愿意再这样淋一场雨。我似乎更偏爱坐在一棵高高撑满的树下，让一阵大风吹彻我曾单薄的身躯，让我像那成千上万哗啦哗啦随风坠落的叶子一样，在那短暂而漫长的时刻体验到我丧失已久的无畏与无拘。然后我就能自然而然若无其事地走开，而不用像那个雨天一样，拖着湿漉漉的身躯，扛着湿漉漉的自行车，爬上五楼台阶，放车，脱衣，洗澡，穿上新衣服。

我一日比一日更讨厌麻烦，然而总是在不停地自行创造许多毫无必要的麻烦。实际上比起理智所希望的风，我更多时候选择顺应情绪主动闯入那明知危险的乌云笼罩的雨区，然后一次又一次地承担自己的选择导致的后果，即使从来没有后悔过。既然是自己做出的选择，便没有后悔的动机，这或许是我笃信的箴言，即使明明是一针麻醉剂而已。

我的回忆录兴许便将围绕上述内容展开。在正式进入回忆之前，我首先要不负责任地提出一些观念。我是一个没什么大众意义上的高考情怀的人。我是一个极端自负且自卑的人。目前的境况虽然看上去不错，但请允许我声称这不过是我定位中的最底线————我行走在预期的道路上，却历经了至多的波折。如果你接受了这些观念，那么欢迎你继续向下读；如果你没有接受，也没有关系，因为这不过是一些我在后文中随时可能推翻的言辞而已。

为了保护当事人的隐私，回忆录的内容将经过一些加工处理。回忆录的每一篇将以“【十八号·中文数字】题目”的格式呈现，你可以方便地参考本博客的链接形式来访问对应博文，但我不会公开每一篇的具体名称。你当然可以通过 Github 仓库方便地查阅我上传了哪些篇目，但我不希望你这样做，这不过是一纸君子协定罢了。

风吹了起来。
